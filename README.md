# Schwab Video Training Samples

Static training-site template built for hosting YouTube-based training series.

- Landing hero with call-to-action
- Three-column responsive grid of video trainings
- Full-screen modal YouTube player with autoplay
- Light / dark mode toggle
- Language toggle (English, French, Spanish, Arabic)
- Google Sheets + Apps Script backend for simple analytics
- Minimal dashboard page that reads summary stats from Sheets

To use, upload these files into the root of the `Schwab-Video-Training-Samples` GitHub repository
and enable GitHub Pages on the `main` branch.
